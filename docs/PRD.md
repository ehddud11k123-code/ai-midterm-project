## 1. 개요 (Overview)

### 1.1 제품 목적
사용자가 텍스트 문서를 붙여넣거나 파일로 업로드하면, 자연어 처리(NLP) 기술을 활용해 문서를 자동으로 분석하고 시각적으로 결과를 제공하는 웹 애플리케이션.

### 1.2 배경
텍스트 문서의 핵심 내용을 빠르게 파악하고 싶은 사용자를 위해, 별도의 AI API 호출 없이 로컬 NLP 라이브러리만으로 즉각적인 분석 결과를 제공한다.

### 1.3 목표
- 문서를 입력하면 5초 이내에 분석 결과를 출력
- 비전공자도 직관적으로 사용할 수 있는 UI 제공
- 데모 환경에서 인상적인 시각 결과물 출력

## 2. 사용자 (Target Users)

| 사용자 유형 | 설명 |
|------------|------|
| 학생 | 논문, 보고서 등 긴 문서를 빠르게 파악하고 싶은 학생 |
| 직장인 | 회의록, 기획서 등을 요약·분석하고 싶은 직장인 |
| 연구자 | 텍스트 데이터의 패턴을 시각화하고 싶은 연구자 |

---

## 3. 기능 요구사항 (Functional Requirements)

### 3.1 핵심 기능 (Must Have)

| ID | 기능 | 설명 |
|----|------|------|
| F-01 | 텍스트 입력 | 텍스트 직접 붙여넣기 또는 .txt 파일 업로드 |
| F-02 | 기본 통계 | 단어 수, 문장 수, 문단 수, 평균 문장 길이 표시 |
| F-03 | 키워드 추출 | 문서에서 상위 10~20개 핵심 키워드 추출 및 빈도 시각화 |
| F-04 | 감정 분석 | 텍스트의 긍정/부정/중립 감정 점수 표시 |
| F-05 | 워드클라우드 | 단어 빈도 기반 워드클라우드 이미지 생성 및 출력 |
| F-06 | 가독성 점수 | Flesch Reading Ease 등 가독성 지표 계산 및 등급 표시 |

### 3.2 부가 기능 (Nice to Have)

| ID | 기능 | 설명 |
|----|------|------|
| F-07 | 문장 길이 분포 | 문장별 길이 히스토그램 시각화 |
| F-08 | 결과 다운로드 | 분석 결과를 JSON으로 다운로드 |
| F-09 | 한국어 지원 | 한국어 텍스트 분석 지원 (KoNLPy 활용) |

## 4. 비기능 요구사항 (Non-Functional Requirements)

| 항목 | 요구사항 |
|------|---------|
| 성능 | 1만 단어 이하 문서 기준 5초 이내 분석 완료 |
| 호환성 | 최신 Chrome, Firefox, Edge 브라우저 지원 |
| 배포 | Streamlit Cloud 또는 로컬 실행 가능 |
| 외부 API | 런타임 중 외부 AI API 호출 없음 (오프라인 동작 가능) |
| 의존성 | Python 3.9+ 환경에서 동작 |

---

## 5. 사용자 흐름 (User Flow)

```
1. 사용자가 웹 페이지 접속
2. 텍스트를 직접 입력하거나 .txt 파일 업로드
3. [분석하기] 버튼 클릭
4. 로딩 스피너 표시 (분석 중)
5. 결과 페이지 출력:
   - 기본 통계 카드
   - 키워드 빈도 바 차트
   - 감정 분석 게이지
   - 워드클라우드 이미지
   - 가독성 점수 및 등급
6. (선택) 결과 다운로드
```

---

## 6. 성공 지표 (Success Metrics)

- 분석 결과가 5초 이내에 출력됨
- 모든 핵심 기능(F-01 ~ F-06)이 오류 없이 동작함
- 데모 시연 시 시각 결과물이 명확하게 표시됨
